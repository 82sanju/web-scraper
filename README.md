# Web Scraper for E-commerce Products

A robust, modular web scraping framework designed to extract product information from e-commerce websites.

## Features

- 🚀 Asynchronous scraping with rate limiting
- 🔄 Automatic retry logic for failed requests
- 🎭 Random user-agent rotation
- 💾 Multiple storage options (CSV, JSON, Database)
- 🧪 Comprehensive test suite
- 📊 Data parsing and validation
- 🔧 Highly configurable via environment variables
